# Influencer Project

MVP user flows and user stories for an influencer-brand contract &amp; escrow platform.

## What's here

- **`user-flows.html`** — single-file, self-contained document with:
  - End-to-end journey overview
  - Six main jobs, each laid out as **Main Job → User Stories → Flow Diagram**
    1. Create &amp; Share Contract (Creator)
    2. Brand Reviews &amp; Pays into Escrow
    3. Pre-Approval &amp; Submit Delivery
    4. Brand Reviews &amp; Payment Releases
    5. Resolve a Dispute
    6. Handle Edge Cases
  - Full status lifecycle as a state diagram

## How to view

Open `user-flows.html` in Chrome (or any modern browser). Diagrams render via Mermaid.js pulled from a public CDN — no build step, no install.

## Source

Built from the MVP "User Journey" brief, preserving the core escrow-and-approval logic, 3-day review window, and bilateral-confirmation dispute model.
